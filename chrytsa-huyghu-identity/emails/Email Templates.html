Cómo integrar.

Estos archivos son HTML puro sin dependencias. Carga el contenido en tu ESP favorito y reemplaza variables con su motor de plantillas. Snippets de ejemplo para los proveedores más usados en RD:

Mailgun · cURL REST

# Mailgun simple send
curl -s --user 'api:YOUR_API_KEY' \\
  https://api.mailgun.net/v3/huyghusrl.com/messages \\
  -F from='HUYGHU <notificaciones@huyghusrl.com>' \\
  -F to='cliente@empresa.do' \\
  -F subject='IT-1 Octubre 2026 enviado a DGII' \\
  -F html="$(cat 03-declaracion-enviada.html)"

SendGrid · Node.js SDK

const sgMail = require('@sendgrid/mail');
sgMail.setApiKey(process.env.SENDGRID_KEY);

const html = fs.readFileSync('04-informe-mensual.html', 'utf8')
  .replaceAll('[Cliente Nombre]', cliente.nombre)
  .replaceAll('[Empresa]', cliente.empresa);

await sgMail.send({
  to: cliente.email,
  from: 'notificaciones@huyghusrl.com',
  subject: `Informe mensual ${periodo}`,
  html,
});

AWS SES · Python (boto3) SDK

import boto3
ses = boto3.client('ses', region_name='us-east-1')

with open('02-recordatorio-dgii.html') as f:
    html = f.read().format(
        cliente_nombre=cliente.name,
        empresa=cliente.razon_social,
        tipo_declaracion='IT-1',
        fecha_vencimiento='20/11/2026',
        dias_restantes='6',
    )

ses.send_email(
    Source='notificaciones@huyghusrl.com',
    Destination={'ToAddresses': [cliente.email]},
    Message={
        'Subject': {'Data': '⚠ Próximo vencimiento DGII'},
        'Body': {'Html': {'Data': html}}
    }
)

SMTP genérico · PHP PHPMailer

use PHPMailer\PHPMailer\PHPMailer;
$mail = new PHPMailer(true);
$mail->isSMTP();
$mail->Host = 'smtp.huyghusrl.com';
$mail->SMTPAuth = true;
$mail->Username = 'notificaciones@huyghusrl.com';
$mail->Password = getenv('SMTP_PASS');
$mail->Port = 587;

$html = file_get_contents('01-welcome.html');
$html = str_replace(
  ['[Cliente Nombre]', '[Empresa]'],
  [$cliente->nombre, $cliente->empresa],
  $html
);

$mail->setFrom('notificaciones@huyghusrl.com', 'HUYGHU');
$mail->addAddress($cliente->email);
$mail->Subject = 'Bienvenido a HUYGHU';
$mail->isHTML(true);
$mail->Body = $html;
$mail->send();
