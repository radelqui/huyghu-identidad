Reglas de Uso

  • Fondo canonico = #FFFFFF (blanco puro). Para variantes premium oscuras usar #27426E, nunca negro puro.
  • Verde #248444 es el color del logo y CTAs. No usar en body text largo.
  • Azul #27426E es el color estructural: contornos, taglines, divisores, textos.
  • Contraste minimo AA: azul sobre blanco = 7.5:1 ✓ · verde sobre blanco = 5.3:1 ✓
  • Para impresion: Pantone 2767 C (azul) + Pantone 349 C (verde). Touche Cover 350g.
  • No combinar verde con amarillos, naranjas o dorados. La paleta es bicolor: azul + verde sobre blanco.
