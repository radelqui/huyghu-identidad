9.2

Cómo editar los archivos SVG.

▶ Figma

Editar en Figma

Arrastre el archivo .svg directamente al canvas de Figma. La aplicación importa los objetos como capas nombradas: bg, headline, body, cta, logo aparecen en el panel izquierdo listos para seleccionar.

Para cambiar texto, doble-click sobre el nodo de texto correspondiente. Para mantener la consistencia tipográfica, instale las fuentes Cinzel, Inter y Cormorant Garamond desde Google Fonts antes de abrir el archivo.

Al exportar, use Format PNG @2x para LinkedIn y PNG @1x para Instagram. Mantenga las dimensiones canónicas: cualquier escalado fuera de proporción degradará la lectura tipográfica.

▶ Illustrator

Editar en Illustrator

Abra el SVG con File → Open. Illustrator preservará todos los gradientes, defs y símbolos. El panel Layers mostrará la jerarquía de objetos nombrados; expanda para acceder a cada elemento individual.

Las plantillas usan el gradiente oro canónico definido en <linearGradient id="g">. Si necesita modificar el oro, edite los stops en el panel Gradient — esto propagará el cambio a todos los elementos que referencian url(#g).

Exporte con File → Export → Export As → PNG. Active "Use Artboards" para que las dimensiones se respeten. Para impresión use 300 DPI; para redes sociales 72 DPI es suficiente.

▶ Inkscape

Editar en Inkscape

Inkscape es la opción gratuita y nativa SVG. Abra el archivo directamente — todos los objetos serán editables. Use el panel Objects (Object → Objects) para navegar la estructura por nombres.

Para reemplazar las fuentes use Text → Text and Font. Si no tiene Cinzel instalada, Inkscape sugerirá una fuente sustituta; recomendamos descargar Cinzel gratis desde Google Fonts antes de editar para preservar la identidad visual.

Exporte con File → Export → Export As PNG. Configure el DPI a 96 para web y 300 para impresión. La opción Document respeta las dimensiones exactas del SVG; úsela siempre.
